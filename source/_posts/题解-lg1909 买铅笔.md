[lg1909传送门](https://www.luogu.org/problem/P1909)

<!-- more -->

#### 算法为模拟。。。

本来这题没那么简单，但是题目规定了只有**三种包装且只卖同一种**，题目就变得简单了！

所以只需算出三种铅笔分别的价格再求**最小值**就完事了。

代码如下：

```cpp
#include <bits/stdc++.h>
using namespace std;
int n;
int a[4][3]; //储存数量与价格
int t1,t2,t3; 
int p1,p2,p3,ans;
int main()
{
	cin>>n;
	for(int i=1;i<=3;i++) //输入
		cin>>a[i][1]>>a[i][2];	
	while(t1<n) //3种情况分别列举
	{
		t1+=a[1][1];  //数量
		p1+=a[1][2];  //价格
	}
	while(t2<n) //条件循环数量必须小于规定数量才循环 
	{
		t2+=a[2][1];
		p2+=a[2][2];
	}
	while(t3<n)
	{
		t3+=a[3][1];
		p3+=a[3][2];
	}
	ans=min(min(p1,p2),p3); //用最小值函数求最小值
	cout<<ans; //输出最小值
}
```
