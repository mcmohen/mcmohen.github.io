# 分割线篇

<!-- more -->

#### 分割线可以用于区分段落~~(也就只能区分段落)~~

- ### 格式：`---`（三个或三个以上`-`号或`*`号）

  ```
  ## 一
  
  心口呀莫要这么厉害地跳，
  
  灰尘呀莫把我眼睛挡住了。
  
  手抓黄土我不放，
  
  紧紧儿贴在心窝上。
  
  几回回梦里回延安，
  
  双手搂定宝塔山。
  
  千声万声呼唤你
  
  ——母亲延安就在这里！
  
  杜甫川唱来柳林铺笑，
  
  红旗飘飘把手招。
  
  白羊肚手巾红腰带，
  
  亲人们迎过延河来。
  
  满心话登时说不出来，
  
  一头扑进亲人怀。
  
  （后省略）
  
  ---
  
  原文来自，诗歌《回延安》，贺敬之作。
  ```
