【题目描述】

给定一个长度为 n 的序列 a[n]，和 q 个修改每个修改给定 l, r, k，代表将 a[l],a[l+1], …,a[r] 全部加上 k。最后输出整个序列。

【输入格式】

第 1 行一个整数 n

第 2 行 n 个数表示初始序列 a

第 3 行一个整数 q 表示询问个数

第 4 行到第 q + 3 行每行三个整数 l, r, k 描述一个修改。

【输出格式】

一行 n 个整数表示最终的序列。
